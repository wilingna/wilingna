## 🧭 What I Build · 我在做什么

Not prompts. Not single tools. **Systems.**
不是 prompt，不是单点工具。**是系统。**

```
Most people use AI:    Ask → Answer
                       问 → 答

I build AI systems:    Define → Simulate → Compare → Decide → Execute
                       定义 → 推演 → 比较 → 决策 → 执行
```

Three things I keep building toward:
我反复在搭的三件事：

- 🏭 **Content Systems** — multi-agent pipelines that produce, not assist · 多 Agent 流水线，**生产**而不是辅助
- 🧠 **Decision Systems** — simulate multiple futures before you commit · 在你投入之前，先把多种未来推演一遍
- 🧰 **AI-Native Products** — turning workflows into shippable tools · 把工作流变成真能用的产品

## 🌟 Featured Systems · 代表作品

### 🏭 [PPTFlux](https://github.com/wilingna/PPTFlux) — 4-Agent Presentation System

> Notes → understanding → structure → expression → interactive HTML deck.
> 资料 → 理解 → 结构 → 表达 → 交互式 HTML 幻灯片。

4 specialized agents in a single web page. Output is editable in **two layers** (writing layer + raw HTML). 6 design languages × 18 layout variants.
一个网页里 4 个专业 Agent。两层可编辑（文案层 + HTML 层），6 套设计 × 18 种排版。

**What's different**: not a Gamma clone, not a template engine. Real agent pipeline with engineering safety nets (`sanitizeAgentSlide`, `slotMap`, `retrySinglePage`).
**差异点**：不是 Gamma 套壳，不是模板引擎。真 Agent 流水线 + 工程化兜底。

### 🤖 [ai-content-pipeline (FLUX)](https://github.com/wilingna/ai-content-pipeline) — 7-Agent Content Factory

> One command → trending topic detected, scripted, peer-reviewed, published-ready.
> 一条命令 → 抓热点、写脚本、多模型审稿、出 4 平台成品。

The standout feature: a **3-model editorial board** (Claude scores, Gemini critiques, GPT decides). Below threshold → automatic rewrite, up to 2 rounds.

5 个层次、5 个工具、一份覆盖 3 条路径的风险地图。**为"做错了真有代价"的决策而生**。

## 🧠 Working Philosophy · 工作哲学

```
"AI is not a tool. It's a system you design."
"AI 不是工具，是你设计出来的系统。"
```

A few principles that show up in everything I build:
反复出现在我所有项目里的几个原则：

- **One job per agent.** No agent does two things well. · 一个 Agent 只做一件事。
- **Structured handoffs over free-form context.** JSON > paragraph. · 用结构化交接代替开放上下文。
- **Engineering safety nets, not prompt magic.** Retries, validators, fallbacks. · 工程兜底，不是靠 prompt 显灵。
- **The user keeps the keys.** AI's output is a starting point, never the ending point. · 用户永远握着钥匙。AI 的产出是起点，不是终点。
